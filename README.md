# CodeAlpha - Basic Network Sniffer

## 📌 Project Overview

This project is developed as part of the CodeAlpha Cyber Security Internship.

The Basic Network Sniffer captures live network packets and displays useful information such as:

- Source IP Address
- Destination IP Address
- Protocol (TCP/UDP/ICMP)
- Source Port
- Destination Port
- Packet Length
- Timestamp

---

## 🛠 Technologies Used

- Python 3.13
- Scapy
- Npcap

---

## ▶️ How to Run

Install Scapy:

```bash
pip install scapy
```

Run the program:

```bash
python network_sniffer.py
```
