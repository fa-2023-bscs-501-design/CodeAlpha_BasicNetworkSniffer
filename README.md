# 🌐 CodeAlpha - Basic Network Sniffer

## 📌 Project Overview

The **Basic Network Sniffer** is a Python-based cybersecurity project developed as part of the **CodeAlpha Cyber Security Internship**.

This application captures live network packets using the **Scapy** library and displays useful network information such as source IP, destination IP, protocol type, and packet length.

---

## 🎯 Objectives

- Capture live network traffic.
- Analyze IP packets.
- Understand TCP/IP communication.
- Learn packet sniffing using Python.

---

## 🚀 Features

- Capture live packets.
- Display Source IP Address.
- Display Destination IP Address.
- Detect Protocol.
- Show Packet Length.
- Real-time monitoring.
- Easy to understand Python code.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/fa-2023-bscs-501-design/CodeAlpha_BasicNetworkSniffer.git
```

Install Scapy

```bash
pip install scapy
```

Run the program

```bash
python network_sniffer.py
```
